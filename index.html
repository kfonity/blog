JSX

    
    Эта строка является JSX

    JSX производит React-элементы

    Реакт учитывает тот факт, что механизм отрисовки связан с ещё несколькими важными механизмам UI: обработка событий,
    изменения состояния со временем и подготовка данных к отображению на экране.

    Вместо того, чтобы отделять технологии, помещяя разметку и логику в отдельные файлы, Реакт разделяет задачи.
    Для этого он использует слабо связанные единицы: "Компоненты", которые содержат и логику, и разметку.

    Реакт не требует использование JSX. Тем не менее JSX даёт абсолютную наглядность при работе с UI внутри кода
    Javascript. Так же он позволяет Реакт показывать полезные предупреждения и сообщения об ошибках.

Встраиваемые выражения в JSX


    В примере ниже мы объявляем переменную с именем name, а затем используем её внутри JSX,
    заключая в фигурные скобки
    

    Можно встроить любое JavaScript выражение в JSX, нужно лишь обернуть его в фигурные скобки.

    Например 2+2, user.firstName, formatName(user) - всё является корректным выражением.

    В следующем примеры мы встраиваем результат вызова JavaScript функции formatName(user)
    в <h1>

    

    Рекомендуется оборочивать JSX выражение в круглые скобки ()

JSX сам является выражением

После компиляции JSX-выражения становятся обычными Javascript-объектами.
Это означает то, что JSX можно использовать внутри оператора if, цикла for, присваивать переменным, принимать как аргумент и возвращать из функций.

Указание атрибутов с помощью JSX

Мы можем использовать ковычки для указания строковых либералов как атрибутов:
Наравне с ковычками, мы можем встроить JavaScript код в атрибут, с помощью фигурных скобок
Не ставить ковычки вокруг фигурных скобок, JSX будет воспринимать Js код, как строковый литерал

Указание потомков с помощью JSX

Если тег пустой, его можно закрыть с помощью />
Однако JSX может иметь потомков

JSX предотвращает атаки инъекцией

В JSX можно встроить любой результат ввода и это будет безопасно!
По умолчанию Реакт избегает обработку любых данных, перед отрисовкой. Таким образом, Мы гарантируем, что не сможем что-то внедрить, написаное не явно. Перед отрисовкой всё преобразуется в строку. Это помогает избежать XSS-атак.

JSX представляет собой JavaScript-объекты

Babel комплирует JSX в вызовы React.createElement().
React.createElement() выполняет несколько проверок, чтобы помочь написать код без багов, но по существу, он создаёт объект типо этого
Данные обкъекты носят название "React-элементы".
